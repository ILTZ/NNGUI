Пункт меню - "Тренировка"
5 слотов для входных значений и одно для целевого (типа float(с плавающей точкой)). Покачто принимаются значения только через точку.
Есть возможность загрузить выборку из файла .txt (см. "ex.txt" в этой директории). Форма ввода значений строго <X.X,X.X,X.X,X.X,X.X\tY.Y> где:
<X.X> - входное значение, <Y.Y> - целевое значение, <\t> - табуляция(через кнопку <Tab>).
Кнопка "Сбросить параметры" возвращает стандартные параметры нейросети и задает случайные свзяи для нейронов.
Стандартные параметры НН:
Количество входных значений - 5
Количество скрытых нодов - 5
Количество выходных значений - 1
Количество циклов обучения - 30000
Количество эпох обучения - 1
Коэффициент обучения - 0.3
///
Пункт меню - "Прогон" 
5 слотов для входных значений (типа float(с плавающей точкой)).
Стандартный опрос нейросети. 
При прогоне без тренировке результат непредсказуем, т.к. веса устанавливаются случайно(позднее будет возможность сохранения весов и конфигураций).
///
Пункт меню - "Настройки"
В ячейки устанавливаются текщие параметры нейросети.
Существует возможность изменения количества связей между входным и 1-м скрытым нейроном, 2-м скрытым и выходным нейроном (устанавливать значения больше 5-ти не рекомендуется,
т.к. приложение, скорее всего, вылетит из-за невозможности перемножения матриц при опросе или тренировке).
При некорректном вводе все значения в блоке будут утеряны (можно переключиться между вкладками, и значения восстановятся).
///
Текущие проблемы:
Приложение работает в однопотоке, и при обучении будет подвисать(в зависимости от параметров обучения). Многопоточность планируется.
Процентная шкала на вкладке "Обучение" не функционирует (исправляется).
Внешний вид убогий, если честно, ищу возможности сделать его приятнее.
Планируется добавления оценки точности нейросети.
